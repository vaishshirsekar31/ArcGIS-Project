Transportation Noise Across the USA using ArcGIS StoryMaps
Overview

This project presents an interactive geospatial StoryMap developed using ArcGIS StoryMaps to analyze and visualize transportation noise across the United States. The StoryMap combines spatial visualization, map-based storytelling, and environmental data analysis to communicate the impact of transportation-related noise pollution through interactive GIS workflows.

The project focuses on improving accessibility and understanding of environmental noise patterns using location-based insights, interactive maps, and narrative-driven geospatial storytelling. ArcGIS StoryMaps is commonly used to combine maps, multimedia, and structured narratives for engaging spatial communication workflows.
